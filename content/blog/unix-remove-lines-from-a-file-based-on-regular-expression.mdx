Just a note to self. Useful for trimming away useless information from a log file for example.

```bash
# Output result
$ sed "/pattern/d" file.log

# Overwrite file
$ sed "/pattern/d" file.log > file.log

# Inplace deletion (requires GNU sed)
$ sed -i "/pattern/d" file.log
```

In the pattern, things like capturing groups and alternations needs to be escaped with a slash, `\`. If you have <a href="http://www.regexbuddy.com">RegexBuddy</a> you can use the <a href="http://www.regexbuddy.com/gnu.html">GNU BRE</a> flavor to help you construct the pattern.
