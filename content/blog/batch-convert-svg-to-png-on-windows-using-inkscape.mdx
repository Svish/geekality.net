Needed to convert several SVGs to PNG, and discovered the free <a href="https://inkscape.org">Inkscape</a> easily can be used in batch scripts.

The following .bat file will convert all SVGs dropped on it to a PNG with height 48 placed next to the original SVG. More options can be found in the <a href="https://inkscape.org/sk/doc/inkscape-man.html">Inkscape manual</a>.

```dos
@echo off
for %%f in (%*) do (
  echo %%~f
  "C:\Program Files\Inkscape\inkscape.exe" ^
    -z ^
    --export-background-opacity=0 ^
    --export-height=48 ^
    --export-png="%%~dpnf.png" ^
    --file="%%~f"

)
```
