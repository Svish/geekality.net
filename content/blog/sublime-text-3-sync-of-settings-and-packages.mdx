On the <b>Original</b>:

1. Move `%appdata%\Sublime Text 3\Packages\User` to a synced folder.
2. Make symbolic directory link from User to synced folder.

On the <b>Other</b>:

3. Replace `%appdata%\Sublime Text 3\Packages\User` with symbolic directory link to the synced folder.

**Source:** <a href="https://packagecontrol.io/docs/syncing#dropbox-windows">packagecontrol.io</a>
